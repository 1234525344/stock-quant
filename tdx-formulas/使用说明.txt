通达信选股公式套装 — 使用说明
===============================

📦 包含 5 个公式文件：

1. 量化多因子评分.tni — 【副图指标】
   综合评分 0-100 分，覆盖 13 个因子
   安装后作为副图指标显示，红黄绿三条线分别标注 A/B/C 级

2. 强势突破选股.tni — 【条件选股】
   突破 20/10/5 日高点 + 放量 + MACD 多头 + 均线多头
   适合做突破跟随策略

3. 放量突破选股.tni — 【条件选股】
   成交量异动 + 价格启动 + 均线确认 + OBV 流入
   适合做量价启动策略

4. MACD金叉启动.tni — 【条件选股】
   金叉信号 + RSI 强势 + 放量确认
   适合做趋势转折策略

5. 均线多头启动.tni — 【条件选股】
   均线多头排列 + 发散加速 + 放量
   适合做顺势跟踪策略


🔧 安装方法（通达信 PC 版）：

方法一：导入公式文件
  1. 打开通达信 → 功能 → 公式系统 → 公式管理器
  2. 点击「导入公式」
  3. 选择对应的 .tni 文件 → 打开
  4. 勾选要导入的公式 → 确定

方法二：手动粘贴
  1. 功能 → 公式系统 → 公式管理器
  2. 选择对应类型：
     - 副图指标 → 「技术指标公式」→ 新建
     - 条件选股 → 「条件选股公式」→ 其他类型 → 新建
  3. 复制 .tni 文件内容粘贴进去
  4. 输入公式名称 → 确定


📖 使用建议：

- 每日收盘后用「条件选股」全市场扫描
- 选出的结果打开「多因子评分」副图看综合得分
- 60 分以上重点跟踪，45 分以上加入观察池
- 结合当日大盘情绪和板块热度一起判断
